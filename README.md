<h2>🚀 Overview</h2>

<p>
ShadowHunt is a next-generation AI system that detects <b>deepfakes, impersonations, and synthetic media</b> in real-time using advanced machine learning and blockchain verification.
</p>

---

<h2>✨ Features</h2>

<table>
<tr>
<td>🧠 <b>AI Detection</b></td>
<td>Detect deepfakes in images, video & audio</td>
</tr>

<tr>
<td>🎭 <b>Impersonation Detection</b></td>
<td>Identify celebrity & executive fraud attempts</td>
</tr>

<tr>
<td>⛓️ <b>Blockchain Verification</b></td>
<td>Immutable watermark validation (Ethereum + IPFS)</td>
</tr>

<tr>
<td>📡 <b>Real-Time Analysis</b></td>
<td>Live camera & WebRTC detection</td>
</tr>

<tr>
<td>📊 <b>Trust Meter</b></td>
<td>Confidence score for authenticity</td>
</tr>
</table>
