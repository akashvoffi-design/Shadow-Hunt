## 🚀 Overview

<div align="center">

<table>
<tr>
<td>

**ShadowHunt** is a next-generation AI-powered platform that detects **deepfakes**, **impersonations**, and **synthetic media** in real-time. Using a multi-algorithm detection engine backed by **computer vision**, **frequency analysis**, and **blockchain-based verification**, ShadowHunt provides a comprehensive shield against digital identity fraud.

> *"In a world of synthetic reality, ShadowHunt restores trust."*

</td>
</tr>
</table>

</div>

<br/>

### 🎯 Why ShadowHunt?

| Problem | ShadowHunt Solution |
|:--------|:-------------------|
| 🎭 Deepfake videos are becoming indistinguishable | Multi-layer AI analysis with **8+ detection algorithms** |
| 🆔 Identity theft using synthetic faces | Real-time **face consistency** & **symmetry analysis** |
| 📷 Manipulated images spread misinformation | **Frequency domain analysis** & **noise pattern detection** |
| 🔓 No way to verify media authenticity | **Blockchain watermarking** with Ethereum + IPFS |
| ⏱️ Existing tools are slow and inaccurate | **2–5 second** analysis with live camera support |

## ⚡ Quick Start

### Prerequisites

| Requirement | Version |
|:------------|:--------|
| Python | 3.8+ |
| pip | Latest |
| Git | Latest |
| Webcam | Optional (for live detection) |

### 📦 Installation

```bash
# 1️⃣  Clone the repository
git clone https://github.com/yourusername/shadowhunt.git
cd shadowhunt

# 2️⃣  Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate        # macOS/Linux
venv\Scripts\activate           # Windows

# 3️⃣  Install dependencies
pip install -r requirements.txt

# 4️⃣  Configure environment variables
cp .env.example .env
# Edit .env with your keys (see Configuration section below)

# 5️⃣  Launch ShadowHunt
python shadowhunt_working.py
```

### 🌐 Open in Browser

```
🔗  http://localhost:5000
```

> [!TIP]
> The web interface includes a futuristic dark-themed dashboard with drag-and-drop file upload and a live camera detection panel.

### ⚙️ Configuration

Create a `.env` file in the project root:

```env
# 🔑 Core Settings
SECRET_KEY=your_secret_key_here
FLASK_ENV=development

# ⛓️ Blockchain (optional)
ETHEREUM_RPC=https://mainnet.infura.io/v3/YOUR_INFURA_KEY
IPFS_GATEWAY=https://ipfs.io/ipfs/
CONTRACT_ADDRESS=0x...

# 🗄️ Database
DATABASE_URL=sqlite:///shadowhunt.db

# 📏 Limits
MAX_CONTENT_LENGTH=52428800   # 50 MB
UPLOAD_FOLDER=uploads
```

## 📡 API Reference

### `POST /api/analyze`

Upload an image for deepfake analysis.

**Request:**
```bash
curl -X POST http://localhost:5000/api/analyze \
  -F "file=@suspect_image.jpg"
```

**Response:**
```json
{
  "type": "image",
  "status": "Likely Real",
  "confidence_real": 0.852,
  "confidence_fake": 0.148,
  "watermark": {
    "found": true,
    "hash": "a3f7c2e91b4d8f06"
  },
  "analysis_details": {
    "face_count": 1,
    "image_dimensions": "1920x1080",
    "analysis_methods": "Advanced Multi-Algorithm Detection",
    "detection_scores": {
      "face_consistency": 0.870,
      "eye_alignment": 0.820,
      "facial_symmetry": 0.790,
      "image_quality": 0.910,
      "color_consistency": 0.860,
      "edge_patterns": 0.750,
      "noise_analysis": 0.880,
      "metadata_check": 0.900,
      "ai_artifacts": 0.830
    }
  }
}
```

### Status Codes

| Code | Meaning |
|:-----|:--------|
| `200` | Analysis complete |
| `400` | Invalid file format |
| `413` | File too large (>50 MB) |
| `500` | Internal server error |

## 📊 Detection Methodology

ShadowHunt employs **9 detection algorithms** running in parallel:

```
 ┌─────────────────────────────────────────────────────────┐
 │              DETECTION PIPELINE (WEIGHTED)              │
 ├─────────────────────┬────────────┬──────────────────────┤
 │ Algorithm           │  Weight    │ Technique            │
 ├─────────────────────┼────────────┼──────────────────────┤
 │ Face Consistency    │   25%      │ LBP Texture Analysis │
 │ AI Artifact Scan    │   20%      │ FFT Frequency Domain │
 │ Eye Alignment       │   20%      │ Haar Cascade + Ratio │
 │ Facial Symmetry     │   15%      │ Mirror Diff Scoring  │
 │ Image Quality       │    8%      │ Laplacian Variance   │
 │ Color Consistency   │    6%      │ HSV Entropy Analysis │
 │ Edge Patterns       │    4%      │ Canny + Contours     │
 │ Noise Analysis      │    1%      │ Gaussian Subtraction │
 │ Metadata Check      │    1%      │ EXIF Tag Inspection  │
 └─────────────────────┴────────────┴──────────────────────┘
```

### Confidence Interpretation

| Score Range | Status | Meaning |
|:------------|:-------|:--------|
| `> 0.70` | ✅ **Likely Real** | High confidence of authenticity |
| `0.30 – 0.70` | ⚠️ **Uncertain** | Requires further analysis |
| `< 0.30` | ❌ **Likely Fake** | Strong deepfake indicators detected |
